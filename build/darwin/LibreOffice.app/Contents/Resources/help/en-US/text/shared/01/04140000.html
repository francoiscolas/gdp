Inserting Images

Inserts an image into the current document with optimal page wrapping and centered on the line at the current cell position centered on the page or slide.

To access this command...

Choose Insert - Image.

On the Standard bar, click

Image


Inserting, Editing, Saving Bitmaps

Frame Style

Select the frame style for the graphic.

Link

Inserts the selected graphic file as a link.

Preview

Displays a preview of the selected graphic file.

Preview Field

Displays a preview of the current selection.
